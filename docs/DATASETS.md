# Public datasets

Raw datasets are not redistributed in this repository. Please obtain them from the original sources and cite them when reusing the experiments.

## Dataset 1: UofM NMC pouch-cell data

- University of Michigan pouch-cell voltage and expansion cyclic aging dataset.
- Cells used in the manuscript subset: Cell01, Cell03, Cell10, Cell11, Cell12.
- Related references used in the manuscript: Mohtat et al., Deep Blue Data, 2021; Mohtat et al., Journal of The Electrochemical Society, 2021.

## Dataset 2: Oxford Battery Degradation Dataset 1

- Kokam lithium-ion pouch cells with graphite anodes and LCO/NCO-blend cathodes.
- Cells used: Oxford cells 1--8.
- Related reference used in the manuscript: Howey and Birkl, University of Oxford, 2017.

## Dataset 3: LFP/graphite dataset

- Dataset associated with real-time personalized battery health prediction.
- Cells used in the manuscript subset: Cell51--Cell55.
- Related references used in the manuscript: Yuan et al., Mendeley Data, 2022; Ma et al., Energy & Environmental Science, 2022.

## Notes

The repository includes derived `features/multi_hi_features.csv` files and executed result files to reproduce the manuscript-level tables. For full end-to-end regeneration from raw data, download the raw datasets from the original providers and update notebook path variables accordingly.
